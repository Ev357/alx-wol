# Atheros ALX Wake-on-LAN kernel module patch
A NixOS-packaged kernel module patch that brings Wake-on-LAN support to the Atheros ALX ethernet driver. 

> [!WARNING]
> **Use at your own risk!**
> This driver is based on the [original 6.13 patch](https://github.com/AndiWeiss/alx-wol) but has been modified **WITH AI** to compile against newer kernel versions. If a patch is available for your kernel version in the original repository, I would recommended using it instead.
